Terminal :

  • Pwd : permet d'afficher le repertoire courant
  • ls : permet d'afficher le contenue d'un répertoire
  • ls -a : l'option -a affiche également les fichier et dossiers cachée
  • ls -l : l'option -l modifie l'affichage pour rajouter de nombreuses info
  • cd dossier : permet de se déplacer à l'interieur d'un répertoire
  • mkdir dossier : permet de créer un dossier
  • touch NomDuFichier : permet de creer un fichier
  • mv source destination : permet de deplacer des éléments
  • * : est un caractère qui peut être utilisé comme substitut pouir n'importe quel caractère dans une recherche
  • cp source destination : permet de copier des éléments
  • cp -r : l'option -r permet de copier un repertoire
  • rm fichier : permet de supprimer des fichiers
  • rm -r dossier : l'option -r permet de supprimer des repertoires
  • man commande : permet d'afficher le manuel d'une commande
  • cat/less/more nomDuFichier : permet d'afficher le contenue d'un fichier
  • > : Permet de rediriger le resultat d'une commande vers un fichier
  • grep ceQueVousVoulezChercher laOuVousVoulezChercher : permet de faire des recherche dans des fichiers
